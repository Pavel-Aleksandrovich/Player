# Player
- Application for playing music

##Branch
- Current branch - **main**

## Features
- Select files from local app 'Files'
- Load files from Document Directory
- Search music 
- Load music from network
- Playing music

## Stack
- Swift
- UIKit
- MVP + Router
- URLSession
- AVFoundation
- FileManager
- UI programmatically (Constraints). Without Storyboard
- DI
